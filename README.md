## Deployment

This website is ready for deployment on Vercel:

1. **Connect to Vercel:**
   - Push your code to GitHub
   - Connect your repository to Vercel
   - Deploy automatically

2. **Manual Deployment:**
   ```bash
   npm install -g vercel
   vercel --prod
   ```
